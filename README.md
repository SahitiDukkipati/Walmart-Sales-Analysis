# 🛒 Walmart Sales Analysis

> A data-driven deep dive into Walmart's retail performance — uncovering sales trends, profit patterns, and store-level insights using SQL, Python, and Power BI.

---

## 📌 Project Overview

This project analyzes Walmart's retail sales data to understand sales performance, profit/loss trends, and store-level behavior. The goal was to transform raw transactional data into actionable insights through a combination of structured querying, exploratory analysis, and interactive dashboards.

---

## 🔧 Tools & Technologies

| Tool | Purpose |
|------|---------|
| **SQL** | Data cleaning, querying, and pattern extraction |
| **Python (Pandas & Matplotlib)** | Trend analysis and profit/loss exploration |
| **Jupyter Notebook** | Interactive data analysis environment |
| **Power BI** | Dashboard design and sales visualization |

## 🚀 What I Did

### 1. 🗄️ SQL — Data Cleaning & Extraction
- Cleaned raw sales data and handled inconsistencies
- Wrote queries to extract **monthly and store-wise sales patterns**
- Aggregated data to prepare it for downstream analysis

### 2. 🐍 Python — Trend & Profit Analysis
- Used **Pandas** for data manipulation and feature exploration
- Built visualizations with **Matplotlib** to analyze sales trends over time
- Identified profit and loss patterns across different periods

### 3. 📊 Power BI — Dashboard & Visualization
- Designed an interactive dashboard showing:
  - **Total Sales** across all stores
  - **Monthly Growth** trends
  - **Top-Performing Stores** ranked by revenue

---

## 📈 Key Takeaways

- 🏪 **Top Stores Identified** — Pinpointed which stores consistently drove the highest monthly sales
- 📉 **Profit/Loss Patterns** — Detected seasonal and operational trends affecting profitability
- 📊 **Clear Growth Insights** — Visualized sales momentum through dashboards for easy stakeholder interpretation
